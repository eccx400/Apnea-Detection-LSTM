This is a pipeline to detect and predict apnea events (either OSA or OSAHS) using a Long-Short Term Memory (LSTM) neural network. The model is trained on positive/negative sequences of breathing data, sampled at 8 hz (8 samples/second). Currently, the three datasets supported are the DREAMS, MIT-BIH, and University of Dublin database.

Steps to run apnea detection pipline

  1. Select excerpt, dataset, apnea type, normalization parameters
  2. Run machine learning model
